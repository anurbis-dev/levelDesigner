# Руководство пользователя - 2D Level Editor v3.52.5

## 🎯 Быстрый старт

1. **Запуск**: Дважды кликните `start_Editor.vbs` (рекомендуется, полностью бесфликерный) или `start_Editor.bat`
2. **Интерфейс**: Canvas (центр), панели (ассеты, детали, слои, структура)
3. **Создание**: Перетащите ассеты из нижней панели на canvas
4. **Группировка**: Выделите объекты → `Ctrl+G`
5. **Слои**: Организуйте объекты по слоям с полной изоляцией видимости
6. **Сохранение**: `Ctrl+S`

## 🆕 Новые возможности v3.51.0

### Унифицированная логика сепараторов и тач-поддержка
- **Дабл-клик на сепараторах**: Единая система сворачивания/разворачивания всех панелей
- **Управление через StateManager**: Все состояния сепараторов сохраняются централизованно
- **Поддержка тач-экранов**: Корректная работа на мобильных устройствах
- **Правильная работа разделителей**: Разделители панелей работают корректно в обеих позициях
- **Исправлено зеркальное движение**: Разделители следуют за пальцем, а не движутся зеркально
- **Унифицированная логика**: Одинаковая работа для мыши и тач-событий

## 🎮 Основные операции

### Выделение объектов
- **Одиночный клик** - выделить объект
- **Shift+клик** - добавить/убрать из выделения
- **Рамка выделения** - перетащить мышью для множественного выбора
- **Скрытые объекты** - не выделяются через canvas (только через Outliner)

### Управление объектами
- **Перемещение**: Перетащить выделенный объект
- **Дублирование**: `Shift+D` или Alt+drag выделения
- **Удаление**: `Delete` или `X`
- **Отмена**: `Ctrl+Z` (Undo), `Ctrl+Y` (Redo)

### Режим редактирования групп
- **Вход**: Двойной клик по группе или `Ctrl+G` при выделении группы
- **Выход**: `Esc` или клик вне группы
- **Alt+drag**: Переместить объект за пределы группы
- **Вложенность**: Поддержка групп внутри групп

## 🎨 Продвинутая система слоев

### Создание и управление слоями
- **Добавить слой**: Кнопка "+" в панели Layers
- **Переименование**: Клик по имени слоя
- **Порядок**: Drag & drop для изменения порядка
- **Цвет**: Клик по цветному индикатору слоя
- **Удаление**: Кнопка "×" (нельзя удалить Main слой)
- **Дублирование**: ПКМ → "Duplicate" для копирования слоя

### Состояния слоёв

**Активные слои** (рамка):
- Слои с выбранными объектами подсвечиваются рамкой
- Цвет рамки настраивается в **Settings → Colors → Selection Colors → Active Layer Border**
- Может быть несколько активных слоёв одновременно
- Подсветка обновляется в реальном времени при выборе объектов

**Мгновенное обновление иконок:**
- Иконки видимости (👁️) и блокировки (🔒) обновляются сразу при клике
- Цвет иконок меняется в зависимости от состояния слоя
- Никаких задержек или необходимости повторного клика

**Текущий слой** (синий фон):
- Слой для создания новых объектов подсвечивается синим фоном
- Только один текущий слой
- Новые объекты создаются в текущем слое
- Клик по слою делает его текущим

### Поиск и фильтрация
- **Поиск**: Поле поиска в верхней части панели слоёв
- **Реальное время**: Результаты обновляются при вводе
- **Регистронезависимый**: Поиск работает без учёта регистра

### Контекстные меню

**ПКМ по слою:**
- **Make Current** - сделать текущим слоем (неактивно для текущего слоя)
- **Rename** - переименовать слой
- **Duplicate** - дублировать слой
- **Hide/Show** - скрыть/показать слой
- **Lock/Unlock** - заблокировать/разблокировать слой
- **Select All Objects** - выбрать все объекты слоя
- **Move Objects to Main Layer** - переместить объекты в Main слой
- **Delete Layer** - удалить слой (неактивно для Main слоя)

**Особенности:**
- **Умное позиционирование** - меню автоматически открывается в видимой области панели
- **Неактивные команды** - недоступные команды отображаются как неактивные вместо скрытия
- **Единообразный дизайн** - все контекстные меню имеют одинаковый стиль сепараторов
- **Ультра-компактный режим** - максимально сжатый интерфейс с размером шрифта 12px и минимальными отступами

**Меню команд (три точки):**
- **Select All Layers** - выбрать все слои
- **Deselect All** - снять выбор со всех слоёв
- **Show All Layers** - показать все слои
- **Hide All Layers** - скрыть все слои
- **Lock All Layers** - заблокировать все слои
- **Unlock All Layers** - разблокировать все слои

### Видимость слоев
- **Hide/Show**: Иконка глаза переключает видимость слоя
- **Полная изоляция**: Скрытые объекты не выделяются через canvas
- **Outliner**: Работает независимо, можно выбирать скрытые объекты
- **Автоочистка**: Selection снимается при скрытии слоя
- **Группы**: Открытые группы закрываются при скрытии слоя

### Статистика слоёв
- **Счётчики объектов**: Показывается количество объектов в каждом слое
- **Обновление в реальном времени**: Статистика обновляется при изменениях
- **Кеширование**: Быстрый доступ к статистике без пересчёта

## 🌌 Эффект параллакса слоев

### Включение параллакса
- Включите **Parallax** в меню **View** → **Object** → **Parallax**
- Параллакс применяется только при включенной опции

### Настройка коэффициентов слоев
- В панели **Layers** вместо иконки удаления теперь числовое поле
- **0** (по умолчанию) - слой не участвует в параллаксе (движется с камерой)
- **Отрицательные значения** (-1, -0.5) - объекты двигаются медленнее камеры
- **Положительные значения** (0.5, 1) - объекты двигаются быстрее камеры

### Установка точки отсчета
- В панели **Level** нажмите **"Set Camera Start Position"**
- Текущая позиция камеры становится референсной точкой
- Эффект параллакса рассчитывается относительно этой точки

### Как работает параллакс
- При перемещении камеры объекты разных слоев двигаются с разной скоростью
- Создается эффект глубины и объема
- Полезно для создания фонов, дальних планов и передних планов
- Параллакс регулируется на двух уровнях: **Per-Layer** (индивидуальный для каждого слоя) и **Level-Wide** (общие множители для всего уровня)

### Per-Layer Parallax Offset (для каждого слоя)
Каждый слой имеет свой `parallaxOffset`. Сдвиг при отрисовке: `cameraOffset × parallaxOffset`, итоговая скорость относительно камеры `1 + parallaxOffset`:
- **Background layer**: parallaxOffset = -0.8 → scroll 0.2 (медленное движение)
- **Middle layer**: parallaxOffset = -0.3 → scroll 0.7 (средняя скорость)
- **Main / no parallax**: parallaxOffset = 0 → scroll 1.0 (как камера)
- **Foreground layer**: parallaxOffset = 0.5 → scroll 1.5 (быстрое движение)
- **UI layer**: parallaxOffset = -1 → scroll 0 (закреплён на экране)

### Level-Wide Parallax Multipliers (множители уровня)
Управляются в DetailsPanel в панели "Camera" (при отсутствии выделения объектов):
- **Parallax H** (Horizontal) — множитель горизонтального параллакса (по умолчанию 1.0, сохраняется с уровнем)
- **Parallax V** (Vertical) — множитель вертикального параллакса (по умолчанию 1.0, сохраняется с уровнем)

Эти множители применяются к смещению камеры независимо по каждой оси и комбинируются с `parallaxOffset` каждого слоя. Например, если установить Parallax H = 0.5, горизонтальный параллакс всех слоёв будет уменьшен вдвое.

### View меню

**Canvas:**
- **Grid** - отображение сетки
- **Game Mode** - скрытие UI панелей
- **Snap To Grid** - привязка к сетке

**Object:**
- **Object Boundaries** - границы объектов (только видимые)
- **Object Collisions** - коллизии объектов (только видимые)
- **Parallax** - эффект параллакса слоев (выключен по умолчанию)

## 🎯 Работа с объектами

### Создание и управление
- **Размещение**: Перетащите ассет из панели на canvas
- **Перемещение**: Перетащите выделенный объект
- **Дублирование**: `Shift+D` с последующим размещением
- **Удаление**: `Delete` или `X`
- **Превью дублирования**: Полупрозрачность 70%

### Группировка
- **Создание**: Выделите 2+ объекта → `Shift+G`
- **Редактирование**: Двойной клик по группе → `Esc` для выхода
- **Alt+drag**: Перемещение объекта за пределы группы
- **Автоочистка**: Пустые группы удаляются автоматически

**Преимущества:**
- Поддерживает чистую иерархию проекта
- Предотвращает накопление "мусорных" пустых групп
- Работает незаметно для пользователя
- Не мешает нормальной работе с группами
- **Полная поддержка вложенных групп** - группы внутри групп любого уровня
- **Корректная сортировка по глубине** - объекты сортируются по полным иерархическим индексам

### Префабы
- Создайте группу → перетащите в панель ассетов → введите имя

## 📊 Панели

### Details
- **При выделении объекта**: свойства Name, Position (X/Y), Size (W/H), Rotation, Color, и дополнительные параметры (Advanced, Custom Properties)
- **Множественное выделение**: отображаются только общие свойства
- **Статистика уровня** когда ничего не выбрано: 
  - Секция **Stats** — количество объектов, групп, типы объектов
  - Секция **Camera** — кнопка "Set Camera Start Position" для сохранения текущей позиции камеры как reference point параллакса, и поле **Parallax H/V** для настройки множителей горизонтального/вертикального параллакса уровня (сохраняются с файлом уровня)
- **Переключение позиции**: Кнопка ⇄ в заголовке для переноса на левую сторону
- **Компактный лейаут**: все поля расположены в одну строку (label справа ~40% ширины + контролы в остатке)

## ⚙️ Настройки

### Colors - Система цветовых настроек интерфейса

#### UI Colors - Цвета пользовательского интерфейса
- **UI Background** - цвет фона интерфейса
- **UI Text Color** - цвет текста в интерфейсе
- **Active Elements** - цвет активных элементов (кнопки, выделения)
- **Active Text Color** - цвет текста активных элементов
- **Active Tab Color** - цвет активных вкладок
- **Accent Color** - цвет акцентов (чекбоксы, радио кнопки, слайдеры)

#### Canvas Colors - Цвета рабочей области
- **Canvas Background** - цвет фона канвы
- **Grid Color** - цвет линий сетки
- **Grid Subdivision** - цвет вспомогательных линий сетки

#### Selection Colors - Цвета выделения объектов
- **Selection Outline** - цвет рамки выделенных объектов
- **Group Outline** - цвет рамки выделенных групп
- **Marquee Selection** - цвет рамки выделения мышью
- **Hierarchy Highlight** - цвет подсветки иерархии
- **Active Layer Border** - цвет границы слоев с выбранными объектами

#### Logger Colors - Цвета категорий логирования
- Настраиваемые цвета для всех категорий логирования (RENDER, UI, FILE, ERROR, etc.)

### UI Settings
- **Compact Mode** - ультра-компактный режим интерфейса:
  - **Размер шрифта** - уменьшен до 12px
  - **Отступы** - минимальные отступы между элементами
  - **Высота элементов** - уменьшенная высота кнопок и панелей
  - **Размеры чекбоксов** - уменьшенные размеры для экономии места

### Grid Settings
- **Grid Type** - выбор типа сетки: Rectangular, Diamond, Hexagonal
- **Grid Size** - размер ячеек сетки (8-512 пикселей)
- **Grid Color** - цвет линий сетки
- **Grid Thickness** - толщина линий (0.1-5.0)
- **Grid Opacity** - прозрачность линий (0-1.0)
- **Hex Orientation** - ориентация хексагонального грида (только для Hexagonal):
  - **Pointy Top** - острые углы сверху/снизу
  - **Flat Top** - плоские стороны сверху/снизу

### Visual Settings
- **Font Scale** - масштаб шрифта для адаптации к экрану
- **Unified Tabs** - единообразный внешний вид всех табов в приложении

### Other Settings
- **Путь к ассетам**: Settings → Assets Path

## ⚙️ Настройки и горячие клавиши

### Горячие клавиши

Все доступные горячие клавиши можно просмотреть в разделе **Settings → Hotkeys**. Основные комбинации:

#### **Общие операции**
- `Esc` - Сбросить выделение объектов (если нет активных процессов) / Отмена текущих действий / Закрыть режим редактирования группы / Закрыть окно настроек
- `Ctrl+Z` - Отменить последнее действие (Undo)
- `Ctrl+Y` или `Ctrl+Shift+Z` - Повторить отмененное действие (Redo)

#### **Работа с объектами**
- `Delete` или `X` - Удалить выделенные объекты
- `Shift+D` - Дублировать выделенные объекты
- `F` - Фокус на выделенных объектах
- `A` - Фокус на всех объектах

#### **Группы и слои**
- `G` - Показать/скрыть сетку
- `Shift+G` - Группировать выделенные объекты
- `Alt+G` - Разгруппировать выделенные объекты
- `PageUp` / `Shift+PageUp` - Поднять объекты на слой выше
- `PageDown` / `Shift+PageDown` - Опустить объекты на слой ниже

#### **Файловые операции**
- `Ctrl+N` - Создать новый уровень
- `Ctrl+O` - Открыть уровень
- `Ctrl+S` - Сохранить уровень
- `Ctrl+Shift+S` - Сохранить уровень как

#### **Дополнительные функции**
- `P` - Переключить параллакс
- `Ctrl+L` - Фокус на поиске слоев
- `Ctrl+Shift+L` - Добавить новый слой

### Кастомизация горячих клавиш

1. Откройте **Settings → Hotkeys**
2. Найдите нужную команду в списке
3. Кликните по полю с клавишей
4. Нажмите желаемую комбинацию клавиш
5. Нажмите **Save Settings** для применения

### Настройки панели Assets

В разделе **Settings → Assets** можно настроить:
- **Размер элементов**: Ctrl+Scroll мышью для изменения размера
- **Режим отображения**: Grid / List / Details (переключается кнопками в панели)

Настройки сохраняются автоматически и восстанавливаются при следующем запуске.

## 🔧 Устранение неполадок

- **Объекты не видны**: Проверьте видимость слоя и объекта
- **Не работает выделение**: Убедитесь, что объект разблокирован
- **Проблемы с производительностью**: Уменьшите количество объектов
- **При ошибках**: Проверьте консоль браузера (F12)
